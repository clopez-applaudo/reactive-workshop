Goal: learn Observable, Subject, operators, switchMap cancellation, shareReplay caching.

Mental model (talk track)

UI events → Subject.next(value) Subject → pipe(operators...) Result Observable → async pipe in template (no manual unsubscribe) switchMap → cancels previous async work when a new value arrives shareReplay(1) → cache latest successful result for new subscribers
